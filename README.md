# To-Do List – Task Management Web Application

## Overview

The To-Do List is a task management web application developed using Spring Boot. It enables users to efficiently create, view, update, and delete tasks through a simple and user-friendly interface. The application demonstrates CRUD operations and follows the MVC architecture.

## Features

- Add New Tasks
- View Task List
- Update Existing Tasks
- Delete Tasks
- CRUD Operations
- Responsive User Interface
- Database Integration with MySQL

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Thymeleaf
- HTML5
- CSS3
- Bootstrap 5
- MySQL
- Maven

## Installation

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Configure MySQL and update the database details in application.properties.
4. Run the Spring Boot application.
5. Open your browser and visit: http://localhost:8080/
